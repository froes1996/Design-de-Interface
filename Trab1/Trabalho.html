Trabalho Design de Interfaces

O que é Design de Interface? O design de interface é o recurso que conduz a interação do ser humano com um produto físico ou virtual. A interface varia de um brinquedo, aparelho eletrodoméstico, até um aplicativo para smartphone ou uma página na web.
